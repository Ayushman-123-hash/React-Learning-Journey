# 📖 About Repository

This repository documents my complete React.js learning journey through hands-on projects while following **Hitesh Choudhary's React.js Playlist**.

Instead of only watching tutorials, I build every project from scratch to strengthen my understanding of React fundamentals, component-based architecture, hooks, state management, routing, API integration, and modern frontend development practices.

The repository will continue to grow as I complete more projects and explore advanced React concepts.

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/Ayushman-123-hash/React-Projects.git
```

Move inside project

```bash
cd React-Projects
```

Install dependencies

```bash
npm install
```

Run Project

```bash
npm run dev
```
